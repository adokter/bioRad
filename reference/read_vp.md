# Read a vertical profile (`vp`) from file

Read a vertical profile (`vp`) from file

## Usage

``` r
read_vp(file)
```

## Arguments

- file:

  A string containing the path to a vertical profile generated by
  [calculate_vp](http://adriaandokter.com/bioRad/reference/calculate_vp.md).

## Value

An object of class
[`summary.vp()`](http://adriaandokter.com/bioRad/reference/summary.vp.md).
