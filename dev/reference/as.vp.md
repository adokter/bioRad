# Convert a dataframe into a vp object

Convert a dataframe into a vp object

## Usage

``` r
as.vp(data)
```

## Arguments

- data:

  a dataframe created from a VPTS CSV file

## Value

a bioRad vp object

## See also

Other profile conversion functions:
[`as.data.frame.vp()`](http://adriaandokter.com/bioRad/dev/reference/as.data.frame.vp.md),
[`as.vpts()`](http://adriaandokter.com/bioRad/dev/reference/as.vpts.md)

## Examples

``` r
# load vp data as a data.frame:
df <- as.data.frame(example_vp)
# convert the data.frame to a vp object:
as.vp(df)
#> Warning: Extra fields found: day, sunrise, sunset
#>                Vertical profile (class vp)
#> 
#>        radar:  seang 
#>       source:  
#> nominal time:  2015-10-18 18:00:00 
#> generated by:   
```
