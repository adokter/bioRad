# Check if radar codes are exactly 5 characters

Check if radar codes are exactly 5 characters

## Usage

``` r
check_radar_codes(radars)
```

## Arguments

- radars:

  character vector. Radar codes to check, e.g. `c("bejab", "bewideu")`.

## Value

NULL. Will stop and show error message if at least one of the provided
radar codes is not exactly 5 characters.
