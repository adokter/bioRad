# Read a vertical profile (`vp`) from UMASS Cajun text file

Read a vertical profile (`vp`) from UMASS Cajun text file

## Usage

``` r
read_cajun(file, rcs = 11, wavelength = "S")
```

## Arguments

- file:

  Character. Path to a text file containing the standard output
  (`stdout`) generated by UMASS Cajun pipeline.

- rcs:

  Numeric. Radar cross section per bird in cm^2.

- wavelength:

  Character or numeric. Radar wavelength, either `C` for C-band (5.3
  cm), `S` for S-band (10.6 cm) or in cm.

## Value

A `vp` object.
