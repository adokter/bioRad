# Vertical profile (`vp`) example

Example of a
[`vp`](http://adriaandokter.com/bioRad/dev/reference/summary.vp.md)
object with name `example_vp`.

## Usage

``` r
example_vp
```

## Format

An object of class `vp` with 25 rows and 16 columns.

## Value

An example object of type `vp` which represents a vertical profile.

## See also

Other vp functions:
[`get_quantity()`](http://adriaandokter.com/bioRad/dev/reference/get_quantity.md),
[`plot.vp()`](http://adriaandokter.com/bioRad/dev/reference/plot.vp.md),
[`summary.vp()`](http://adriaandokter.com/bioRad/dev/reference/summary.vp.md)
